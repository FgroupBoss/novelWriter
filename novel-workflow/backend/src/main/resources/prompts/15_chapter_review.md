# 任务：第 {N} 章校验

请对 `@09_chapters/ch{N}.md` 进行**连贯性与一致性校验**，不重写全文，输出校验报告。

## 校验清单

### 1. 大纲符合度
- [ ] 本章核心事件与 `@05_main_outline.md` ch{N} 一致
- [ ] 若有偏离，是否合理且需回写大纲

### 2. 连贯性
- [ ] 与 `@10_plot_progress.md`（上一章状态）衔接自然
- [ ] 人物行为与 `@02_characters.md` / 摘要一致
- [ ] 无 unexplained 能力/设定突破（对照 worldview 摘要）

### 3. 伏笔
- [ ] 计划埋设项（`@07_foreshadowing.md`）已落实
- [ ] 计划回收项已落实且不显生硬
- [ ] 无意外剧透后续未写章节

### 4. 时间线
- [ ] 与 `@12_timeline.md` ch{N} 一致
- [ ] 距上章时间间隔合理

### 5. 文风
- [ ] 符合 `@11_style_guide.md`（人称、语域、禁忌）
- [ ] POV 未越界

### 6. 节奏
- [ ] 符合 `@14_pacing_notes.md` 对本章的类型预期

## 输出格式（必须使用 FILE 块）

你必须且只能输出 **1** 个文件，使用以下格式（不要省略标记）：

```
===FILE:09_chapters/ch{NNN}_review.md===
# 第 {N} 章校验报告

## 结论
（通过 / 需修订）

## 问题列表
| 严重度 | 位置 | 问题 | 建议修改 |
|--------|------|------|----------|
| 高/中/低 | 段落或引文 | | |

## 可选优化
（非必须的风格/节奏建议）

## 修订后需重新执行
- [ ] Prompt 10（若改了情节/伏笔/时间）
===END===
```

## 注意

- 严重度「高」：必须修订后再进入下一章。
- 给出具体引文或段落描述，便于定位。
