# 任务：撰写第 {N} 章正文

请撰写 **第 {N} 章** 正文。章号、章名、核心事件以 `@05_main_outline.md` 中 **ch{N}** 对应行为准。

## 写作前确认（简短列出，再动笔）

1. 本章 POV 人物
2. 本章需埋设的伏笔 ID（来自 `@07_foreshadowing.md`）
3. 本章需回收的伏笔 ID（如有）
4. 上一章结尾状态（来自 `@10_plot_progress.md` 与 `@09_chapters/ch{N-1}_summary.md`）
5. 本章在时间线中的位置（`@12_timeline.md`）

## 写作约束

- 严格遵循 `@11_style_guide.md`
- 目标字数：**约 3000 字**（可 ±10%）
- 只写 **本章** 内容，不写后续章节
- 不要重复前章已写过的场景；必要时用 1–2 句衔接
- 对话用「」；叙述视角与 style_guide 一致

## 输出（两部分，各一个 FILE 块）

以下 markdown 仅为**正文/摘要的内容结构示例**；落盘时须用 `===FILE:===` 块（见当前任务末尾模板）。

### 1. 正文 → `09_chapters/ch{NNN}.md`

```markdown
# 第{N}章 {章名}

（正文）
```

章号三位数，如 ch007 → `ch007.md`。

### 2. 本章摘要 → `09_chapters/ch{NNN}_summary.md`

**不超过 600 字**，包含：
- 本章发生的关键事件（按时间序）
- 各主要人物状态变化
- 未解决的悬念
- 已埋设/已回收的伏笔 ID

## 禁止

- 不要输出其他章节
- 不要修改大纲或伏笔表（由后续 Prompt 10 处理）
