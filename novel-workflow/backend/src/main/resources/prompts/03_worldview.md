# 任务：生成世界观

请根据 `@00_idea.md` 与已有时代/角色摘要，撰写**世界观设定**，保存到 `03_worldview.md`。

## 输出结构

```markdown
# 世界观

## 世界规则（硬规则）
1. …
（明确、可验证、不可随意打破的规则）

## 地理与势力
- **主要地点**：（名称、特征、叙事用途）
- **势力分布**：（名称、目标、资源、与主角关系）

## 历史大事件（故事开始前）
| 事件 | 时间 | 影响 | 是否将在正文中揭示 |
|------|------|------|-------------------|

## 社会运行逻辑
- 法律/潜规则：
- 冲突解决方式：
- 信息传播方式：

## 特殊设定详解
（根据题材：魔法体系/科技树/怪谈规则等，写清边界与代价）

## 世界观未解之谜
（可供支线与伏笔使用，标注重要程度）
```

## 同时输出摘要

使用独立 FILE 块输出 `meta/summaries/worldview_summary.md`，**不超过 400 字**。

## 注意

- 硬规则与 `@01_era_setting.md` 一致，冲突处标注并建议修正。
- 长篇需预留扩展空间（未探索区域、未登场势力可简要预留）。
