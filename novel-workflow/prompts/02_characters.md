# 任务：生成角色设定

请根据 `@00_idea.md` 与时代设定（`@01_era_setting.md` 或 `@meta/summaries/era_summary.md`），撰写**角色设定**，保存到 `02_characters.md`。

## 输出结构

```markdown
# 角色设定

## 角色总览
| 姓名 | 身份 | 叙事功能 | 首次登场（计划章） |
|------|------|----------|-------------------|

## 主角

### [姓名]
- **基本信息**：年龄、外貌、身份
- **性格**：核心特质 + 缺陷
- **背景**：
- **动机**：想要什么 / 害怕什么
- ** character arc（弧光）**：起点 → 终点
- **语言习惯**：（口癖、措辞风格，供正文用）
- **秘密/未知信息**：（对读者隐藏、计划揭露的章号）

## 核心配角
（每人同上结构，可略简）

## 反派/对立面
（同上；若是势力或系统，写清「人格化代表」）

## 群像与功能性角色
（列表即可：姓名、功能、备注）

## 角色动力冲突矩阵
| 角色A | 角色B | 冲突点 | 预期爆发章（段） |
|-------|-------|--------|-----------------|
```

## 长篇要求

- 主角与核心配角 **3–5 人** 需完整弧光。
- 计划贯穿全书的配角至少 **8–12 人** 列入总览表。
- 功能性角色可后续补充，但主要势力需有代表角色。

## 同时输出摘要

使用独立 FILE 块输出 `meta/summaries/characters_summary.md`，**不超过 400 字**：只含姓名、身份、核心动机、关键关系、弧光方向。
