# 任务：生成时代设定

请根据 `@00_idea.md` 中的创意，撰写完整**时代设定**，保存到 `01_era_setting.md`。

## 输出结构（严格遵循）

```markdown
# 时代设定

## 时空定位
- 年代/纪元：
- 地理范围：
- 与真实历史的差异：（如有）

## 社会结构
- 阶级/阶层：
- 政治体制：
- 经济形态：
- 日常生活特征：

## 科技/魔法/超自然水平
- 能力边界：（能做什么、不能做什么）
- 对日常生活的影响：
- 限制与代价：

## 文化与信仰
- 主流价值观：
- 宗教/信仰/禁忌：
- 语言与礼仪：

## 时代氛围关键词
（3–5 个词，供后续文风参考）

## 与核心冲突的关联
（说明时代背景如何推动或制约 `@00_idea.md` 中的核心冲突）
```

## 同时输出摘要

使用独立 FILE 块输出 `meta/summaries/era_summary.md`，**不超过 400 字**，第三人称，只保留后续阶段必需的事实性信息（勿在 FILE 块外裸输出摘要）。

## 注意

- 长篇尺度：设定需支撑 80 章左右展开，避免过于狭窄难以持续。
- 若 idea 未明确时代，给出 2 个方向并**选定一个**展开（说明选择理由），不要留空。
