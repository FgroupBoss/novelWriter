# 任务：生成长篇主线大纲

请根据 `@00_idea.md` 与各设定摘要，撰写**主线大纲**，保存到 `05_main_outline.md`。

## 篇幅参数

- 目标：**长篇，约 80 章**
- 结构：**4 卷 × 每卷约 20 章**（可按题材微调）
- 每章需有：章号、章名、POV（视角人物）、核心事件、情感节拍、与主线的关系

## 输出结构

```markdown
# 主线大纲

## 全书梗概
（200–300 字）

## 三幕/四卷总览
| 卷 | 章范围 | 幕功能 | 核心转折 |
|----|--------|--------|----------|
| 卷一 | ch001–ch020 | 建置 | |
| 卷二 | ch021–ch040 | 对抗升级 | |
| 卷三 | ch041–ch060 | 中点/低谷 | |
| 卷四 | ch061–ch080 | 高潮/收束 | |

---

## 卷一：[卷名]

### 第一幕（建置）ch001–ch007
| 章 | 章名 | POV | 核心事件 | 情感节拍 | 备注 |
|----|------|-----|----------|----------|------|

### 第二幕上（…）ch008–ch014
…

### 第二幕下 / 卷末高潮 ch015–ch020
…

---

## 卷二：[卷名]
（同上格式，每章一行表格）

## 卷三：[卷名]
…

## 卷四：[卷名]
…

## 主线里程碑
| 事件 | 计划章 | 类型（转折/ revelation / 高潮） |
|------|--------|----------------------------------|
```

## 同时输出摘要

使用独立 FILE 块输出 `meta/summaries/main_outline_summary.md`，**不超过 400 字**，只保留各卷核心走向与关键转折点章号。

## 注意

- 必须覆盖 `@00_idea.md` 中的核心冲突与预期结局方向。
- 每章「核心事件」一句话，便于正文阶段按需检索。
- 长篇避免前 20 章纯日常无推进。
