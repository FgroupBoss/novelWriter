# 第1章摘要

**关键事件（按时间序）：**

1. 沈符在符律司处理四十七份案卷（符纸申领、禁术举报、品级复核），眼下一叠卷宗未批完。
2. 符卫捉住翻墙偷符纸的流民少年阿九（黑户，约11–12岁），沈符问话后放其离去，指示出城捷径，未按律断指。
3. 沈符在批阅间隙出现孟寒江封印的闪回片段（碎雪原石殿、铁锁链、对方最后一句“你做了选择”），被震醒后笔落溅朱砂。
4. 薛明夷晚访，告知碧潮州朱砂调运量被魏咸压至十五斤（实际需二十斤）；并点明：符纸消耗量逐年递增，库存将竭。
5. 沈符独自坐在大堂，再度被闪回碎片笼罩；最终理好卷宗，关门离去。

**主要人物状态变化：**

- **沈符**：疲惫、无力；放走阿九是下意识善念也是内心柔软；闪回表明孟寒江封印是他未愈合的创伤；对“符纸将尽”的现实有清醒认知但无解。
- **阿九**：被放走后消失于暮色中，尚未与主线产生直接关联（埋设传承线索）。
- **薛明夷**：短暂出现，带来朱砂短缺与魏咸施压的内幕，语气克制但暗含对魏咸的不满，处于体制内边缘的观测姿态。
- **魏咸**（未亲自出场）：通过薛明夷的话及老工匠的传闻体现其压制与独断（削减朱砂调运、制造符纸短缺）。

**未解决的悬念：**

1. 孟寒江封印的具体过程仍未揭示（闪回碎片仅交代“沈符主动封印了对方”）。
2. 阿九获得符纸后的去向与后续遭遇未知。
3. 符纸与朱砂的紧缺将持续升级，成为沈符行动的外部核心限制。
4. 魏咸削减朱砂的动机未明（是节约还是有意压制沈符一方？）。

**已埋设/已回收的伏笔：**

- **F001（阿九偷符纸）**：已埋设。阿九出场、沈符放其离去、指尖压纸的小心姿态为后文觉醒与传承埋下伏线。
- **F003（孟寒江封印闪回）**：已埋设。本章共三次闪回碎片（大堂批卷时、放走阿九后、独坐闭眼），频率递增，暗示沈符对此事的恐惧与压抑。

**情感基调：** 疲惫无力与压抑中的细微暖意（放走阿九）。全章沉静、克制，以符纸将尽与记忆碎片勾勒出末法时代的基本氛围。
